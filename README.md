# Titulación

App instalable de tarjetas de estudio para enfermería. Incluye sesiones de repaso, exámenes, progreso guardado en el dispositivo y minijuegos desbloqueables.

## Abrirla

Desde esta carpeta, ejecuta:

```powershell
python -m http.server 8080
```

Después abre `http://localhost:8080` en Chrome o Edge. En el menú del navegador elige **Instalar Titulación** para usarla como app.

El progreso se guarda localmente y, después de la primera visita, la app funciona sin conexión.
